Rules for the game!

  • Two players take turn of marking empty tiles
  • Player X starts the game by putting an X in one of the tiles
  • Player O continues by doing the same in a different tile
  • First one to have three in a row has won the game!
  • If all tiles are marked without anyone winning, it's a Tie!
